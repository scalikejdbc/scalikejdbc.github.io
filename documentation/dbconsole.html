Setup

For initial setup instructions, please refer to the /documentation/setup.

Usage

Once installed, the dbconsole command allows easy database connection. Here’s how to use it:

$ dbconsole -h

dbconsole is an extended sbt console to connect database easily.

Usage:
  dbconsole [OPTION]... [PROFILE]

General options:
  -e, --edit    edit configuration, then exit
  -c, --clean   clean sbt environment, then exit
  -h, --help    show this help, then exit

Try sandbox db
